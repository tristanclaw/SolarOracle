## Quick Start

1. Clone the repo:
   ```bash
   git clone https://github.com/tristanclaw/SolarOracle.git
   cd SolarOracle
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run locally:
   ```bash
   npm run dev
   ```

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
