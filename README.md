# The Solar Oracle - Landing Page

A high-conversion GitHub landing page for the Solar Oracle project.

## Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Setup

1. Install dependencies:
```bash
npm install framer-motion lucide-react
```

2. Import and use the component:
```jsx
import SolarOracle from './LandingPage';

function App() {
  return <SolarOracle />;
}
```

## Design Notes
- **Theme:** Rugged Cyberpunk / Humanitarian Aid
- **Colors:** Slate 900, Amber 500, Concrete Grey
- **Font:** Sans (Inter/Roboto), Mono (JetBrains Mono)
